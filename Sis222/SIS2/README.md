# Парсинг данных об аренде квартир с Krisha.kz

## Обзор проекта
Этот проект реализует полный пайплайн данных, который парсит объявления об аренде квартир с krisha.kz, очищает данные и загружает их в базу данных SQLite. Весь процесс автоматизирован с помощью Apache Airflow.

## Описание сайта
Krisha.kz - это ведущая платформа недвижимости в Казахстане с динамическим JavaScript-контентом, бесконечной прокруткой и интерактивным интерфейсом.

## Структура проекта

- `database.py` - создание схемы SQLite
- `krisha_dag.py` - определение DAG Airflow для пайплайна
- `requirements.txt` - зависимости проекта
- `README.md` - это файл
- `src/cleaner.py` - очистка и валидация данных
- `src/loader.py` - загрузка очищенных данных в БД
- `src/scraper.py` - скрапинг объявлений с krisha.kz

## Быстрый старт

1. Установите зависимости:

```bash
pip install -r requirements.txt
```

2. Создайте папку для данных и схему БД:

```bash
mkdir -p data
python database.py
```

3. Запустите скрапер/очистку/загрузку вручную (опционально):

```bash
python src/scraper.py
python src/cleaner.py
python src/loader.py
```

## Замечания
- Убедитесь, что у вас установлены Chrome и совместимый Chromedriver (пакет `webdriver-manager` помогает с этим).
- Для Airflow-запуска требуется установленный и настроенный Apache Airflow.
